JB3D Vendas v3.2 — Gráfico financeiro diário 1. Envie todos os arquivos desta pasta para a raiz do GitHub Pages. 2. Abra limpar-cache.html uma vez após substituir os arquivos. 3. Abra Painel > Gráficos > Financeiro. 4. O novo gráfico compara por dia: vendas, gasto com ADS, custo dos produtos e lucro estimado. 5. Toque ou clique em qualquer data para ver os quatro valores no balão. Os dados, custos e configurações continuam armazenados no navegador.
